# stayhub-listings-app
A full-stack Airbnb-style listings CRUD app built with Node.js, Express, MongoDB, and EJS.

*Features
. Create new property listings
. View all listings
. Edit existing listings
. Delete listings
. Location & country support
. Price management
. Clean and responsive UI

* Tech Stack
. Frontend: EJS, HTML, CSS
 .Backend: Node.js, Express.js
.Database: MongoDB
. ODM: Mongoose
